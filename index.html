SAP API - DOCUMENTAÇÃO

Encontra-se abaixo todas as rotas da SAP API. Até o momento, temos apenas as funcionalidades básicas de cadastrar, listar, atualizar e deletar.

Em um futuro próximo serão adicionadas as rotas para as atualizações de etapas das aquisições, além do sistema de lista negra e de controle de usuários

O sistema ainda não possui validação de dados ou segurança implementada.

Por favor, seja gentil com a SAP API.

Requisições POST:

Para todas as requisições POST, é enviado apenas um json contendo todos os atributos.

Para todas as outras requisições pode ser enviado um json caso haja entrada de dados.

Para o caso onde não é necessário o envio de nenhum dado, basta chamar a rota sem passar nada.
