SAP API - DOCUMENTAÇÃO

Encontra-se abaixo todas as rotas da SAP API. Até o momento, temos apenas as funcionalidades básicas de cadastrar, listar, atualizar e deletar.

Em um futuro próximo serão adicionadas as rotas para as atualizações de etapas das aquisições, além do sistema de lista negra e de controle de usuários

O sistema ainda não possui validação de dados ou segurança implementada.

Por favor, seja gentil com a SAP API.

ROTAS SETORES

Tipo de requisição Caminho Atributos retornados/necessários (tipo de dado)
GET setores /setores

status (string), message (string), data (array de objetos),

atributos dos objetos contidos em 'data': _id (ObjectId, conversível para string), nome (string), email (string), ramal (number), sigla (string), create_date (date)

POST setor /setores nome (string), email (string), ramal (string), sigla (string)
GET setor /setores/:setor_id

message (string), data (objeto)

atributos do objeto 'data': _id (ObjectId, conversível para string), nome (string), email (string), ramal (number), sigla (string), create_date (date)

PUT setor /setores/:setor_id nome (string), email (string), ramal (string), sigla (string)
PATCH setor /setores/:setor_id nome (string), email (string), ramal (string), sigla (string)
DELETE setor /setores/:setor_id -

ROTAS AQUISIÇÕES

Tipo de requisição Caminho Atributos retornados/necessários (tipo de dado)
GET aquisições /aquisições

status (string), message (string), data (array de objetos),

atributos dos objetos contidos em 'data': _id (ObjectId, conversível para string), numAquisicao (string), etapa (number), solicitante (ObjectId), nProcessoSEI (string), objeto (string), modalidade (string), valorTotal (number), tipo (string), orcamento (string), dataAbertura (date), dataEntregaItens (date), fornecedores (array de ObjectIds), fornecedoresVencedores (array de ObjectIds), itens (array de objetos), empenhos (array de objetos), pagamentos (array de objetos)

atributos dos objetos contidos em 'itens': _id (ObjectId), adquirido (boolean), aprovado (boolean), nome (string), unidade (string), quantidade (number), valorUnitario (number), valorTotal (number), catmat_catser (string)

atributos dos objetos contidos em 'empenhos': _id (ObjectId), create_date (date), numero (int), fornecedor (ObjectId), valor (number), itens (array de ObjectIds)

atributos dos objetos contidos em 'pagamentos': _id (ObjectId), dataEnvioPagamento (date), dataPagamento (date), numeroOB (string), numNotaFiscal (string)

POST aquisicao /aquisicoes

solicitante (ObjectId), nProcessoSEI (string), objeto (string), modalidade (string), valorTotal (number), tipo (string), orcamento (string), fornecedores (array de objetos), itens (array de objetos)

atributos dos objetos contidos em 'fornecedores': _id (ObjectId)

atributos dos objetos contidos em 'itens': nome (string), unidade (string), quantidade (number), valorUnitario (number), valorTotal(number), catmat_catser (string)

GET aquisicao /aquisicoes/:aquisicao_id

message (string), data (objeto)

atributos do objeto 'data': _id (ObjectId, conversível para string), numAquisicao (string), etapa (number), solicitante (ObjectId), nProcessoSEI (string), objeto (string), modalidade (string), valorTotal (number), tipo (string), orcamento (string), dataAbertura (date), dataEntregaItens (date), fornecedores (array de ObjectIds), fornecedoresVencedores (array de ObjectIds), itens (array de objetos), empenhos (array de objetos), pagamentos (array de objetos)

atributos dos objetos contidos em 'itens': _id (ObjectId), adquirido (boolean), aprovado (boolean), nome (string), unidade (string), quantidade (number), valorUnitario (number), valorTotal (number), catmat_catser (string)

atributos dos objetos contidos em 'empenhos': _id (ObjectId), create_date (date), numero (int), fornecedor (ObjectId), valor (number), itens (array de ObjectIds)

atributos dos objetos contidos em 'pagamentos': _id (ObjectId), dataEnvioPagamento (date), dataPagamento (date), numeroOB (string), numNotaFiscal (string)

PUT aquisicao /aquisicoes/:aquisicao_id

solicitante (ObjectId), nProcessoSEI (string), objeto (string), modalidade (string), valorTotal (number), tipo (string), orcamento (string), dataAbertura (date), dataEntregaItens (date), fornecedores (array de ObjectIds), fornecedoresVencedores (array de ObjectIds), itens (array de objetos), empenhos (array de objetos), pagamentos (array de objetos)

atributos dos objetos contidos em 'itens': adquirido (boolean), aprovado (boolean), nome (string), unidade (string), quantidade (number), valorUnitario (number), valorTotal (number), catmat_catser (string)

atributos dos objetos contidos em 'empenhos': numero (int), fornecedor (ObjectId), valor (number), itens (array de ObjectIds)

atributos dos objetos contidos em 'pagamentos': dataEnvioPagamento (date), dataPagamento (date), numeroOB (string), numNotaFiscal (string)
